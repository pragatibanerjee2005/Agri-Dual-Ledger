# 🌾 AgriChain

A blockchain-powered agricultural supply chain system that improves transparency, security, and trust between farmers, distributors, retailers, and consumers.

## 🚀 Features
- Dual Ledger Blockchain
- Secure Transactions
- Product Traceability
- Fraud Detection
- Smart Contracts
- Supply Chain Transparency

## 🛠 Tech Stack
- Python
- Blockchain
- Flask
- Machine Learning
- SQLite

## 📂 Project Structure
- blockchain/ → Ledger modules
- data/ → Sample datasets
- notebooks/ → Research & testing
- app.py → Main application

## 📈 Future Scope
- IoT Integration
- Mobile App
- Export Tracking
- Government Marketplace Integration
