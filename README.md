## 📌 Project Overview

Agri-Dual-Ledger is designed to solve common agricultural supply chain problems such as:

* Lack of transparency in product movement
* Fraudulent transactions
* Manual and outdated record systems
* Difficulty in tracking verified cases

This platform combines:

* **Dual-Ledger Blockchain Architecture**
* **AI-Based Fraud Detection**
* **Real-Time Dashboards**
* **Smart Analytics**
